## Props

| Prop           | Type        | Default                 | Description                                |
| -------------- | ----------- | ----------------------- | ------------------------------------------ |
| `blurHeight`   | `number`    | `30`                    | Height of the blur effect in pixels        |
| `blurStrength` | `number`    | `8`                     | Strength of the blur effect (1-20)         |
| `blurColor`    | `string`    | `rgba(26, 26, 26, 0.8)` | Color of the blur gradient                 |
| `children`     | `ReactNode` | Optional                | The content to be wrapped by the component |
